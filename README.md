# IT Support and Troubleshooting Guide

## Overview
This repository contains practical troubleshooting notes and operational guides
based on real-world experience in IT Support roles (L1–L2).

The focus is on end-user support, endpoint troubleshooting, Microsoft 365 administration,
and basic on-premises hardware assistance in managed environments.

## Scope & Role Definition
- Focused on L1–L2 IT Support and operational assistance
- Covers user support, endpoint issues, and tenant-level administration
- Does NOT include system architecture design or server service implementation
- Content is generalized to avoid organization-specific configurations

## Key Areas Covered
- End-user troubleshooting (Windows, macOS, Outlook, SharePoint)
- Microsoft 365 Admin Center operations (Exchange, Entra ID, Groups)
- Endpoint management and imaging workflows
- Hardware support (laptops, printers, docking, physical servers)
- Real-world constraints such as security policies and legacy systems

## Repository Structure
- **general-troubleshooting.md**  
  User-facing issues, application problems, and environment-specific troubleshooting

- **m365-admin-tasks.md**  
  Microsoft 365 tenant-level support tasks and administrative operations

- **windows-endpoint-support.md**  
  Windows imaging, networking, Intune, and endpoint troubleshooting

- **hardware-and-onprem.md**  
  Asset management, hardware setup, and physical server overview
